## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

### Notes

一般情况下, `provider` 是通过 `@Injectable` 声明, 然后在 `@Module` 的 `providers` 数组里注册的 `class`. 灵活运用 `provider` 类型, 可利用 `Nest` 的 `IOC` 容器中注入任何对象.

- 默认的 `token` 就是 `class`, 这样不用使用 `@Inject` 来指定注入的 `token`

- 可使用字符串类型的 `token`, 不过注入的时候要用 `@Inject` 单独指定

- 可以用 useClass 指定注入的 class，还可以用 useValue 直接指定注入的对象

- 可以使用 useFactory，它的参数也注入 IOC 容器中的对象，然后动态返回 provider 的对象

- 可以用 useExisting 给已有的 token，指定一个新 token

## Installation

```bash
$ pnpm install
```

## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Test

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
